

Поля карточки поступления товаров

1. Номер поступления - заполняется автоматически порядковым номером поступления, считаются только сохраненные поступления
2. Дата - по умолчанию становится сегодня, обязательное
3. Название - выбор из списка товаров, обязательное. При выборе должны быть видны: название, артикул
4. Клиент - выбор из списка клиентов, обязательное
5. Количество товаров
6. Цвет - выбор из справочника цветов, появляется, если в поле название выбран товар типа одежда
7. Размер - выбор из справочника размеров, появляется, если в поле название выбран товар типа одежда



Справочники клиентов, цветов, размеров, товаров разместить на вкладке "Справочники", доступном только администратору
