# СТРАНИЦА: СПРАВОЧНИК РАЗМЕРОВ

---

# 1. Цель

Реализовать страницу списка размеров.

Страница предназначена для:

- просмотра списка размеров
- фильтрации
- сортировки
- перехода к карточке
- создания нового размера

---

# 2. Зависимости

Используются:

- LIST PAGE PATTERN
- SYSTEM: Query State Management

---

# 3. Доступ

- доступ только для роли **admin**
- при отсутствии доступа:
  - отображение 403 страницы

---

# 4. Маршрут

/dictionaries/sizes

---

# 5. Конфигурация страницы

Страница реализуется через:

- COMPONENT: FiltersPanel
- COMPONENT: Table
- COMPONENT: Pagination

---

## 5.1 Breadcrumbs

Формируются автоматически:

Главная / Справочники / Размеры

---

## 5.2 FiltersPanel (конфигурация)

### Фильтры:

1. Название размера (name)
   - тип: text
   - placeholder: "Поиск по названию"
3. Актуален (is_active)
   - тип: select
   - значения:
     - Да
     - Нет

---

### Actions:

- CreateButton
  - текст: "Создать"
  - переход:
    /dictionaries/sizes/new

- ResetFilters
  - сброс всех фильтров

---

## 5.3 Table (конфигурация)

### Колонки:

1. Название размера (name)
3. Актуален (is_active)
4. Дата создания (created_at)

---

### Сортировка:

Поддерживается через:

- COMPONENT: Sorting

Доступна по:

- name
- is_active
- created_at

---

### Поведение:

- строка кликабельна
- переход:

/dictionaries/sizes/{id}

---

## 5.4 Pagination

Используется:

- COMPONENT: Pagination

Настройки:

- limit: 20 / 50 / 100
- default: 20

---

# 6. Поведение

---

## 6.1 Загрузка данных

- GET /sizes
- учитывает query state

---

## 6.2 Query параметры

- name
- is_active
- page
- limit
- sort

---

## 6.3 Пустое состояние

- таблица без строк
- сообщение: "Данные отсутствуют"

---

## 6.4 Ошибка

- сообщение ошибки
- retry

---

# 7. API

GET /sizes
