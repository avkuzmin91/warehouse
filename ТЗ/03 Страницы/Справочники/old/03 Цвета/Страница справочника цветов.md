# СТРАНИЦА: СПРАВОЧНИК ЦВЕТОВ

---

# 1. Цель

Реализовать страницу списка цветов.

Страница предназначена для:

- просмотра списка цветов
- фильтрации
- сортировки
- перехода к карточке
- создания нового цвета

---

# 2. Зависимости

Используются:

- LIST PAGE PATTERN
- SYSTEM: Query State Management

---

# 3. Доступ

- доступ только для роли **admin**
- при отсутствии доступа:
  - отображение 403 страницы

---

# 4. Маршрут

/dictionaries/colors

---

# 5. Конфигурация страницы

Страница реализуется через:

- COMPONENT: FiltersPanel
- COMPONENT: Table
- COMPONENT: Pagination

---

## 5.1 Breadcrumbs

Главная / Справочники / Цвета

---

## 5.2 FiltersPanel

### Фильтры:

1. Название (name)
   - тип: text
   - placeholder: "Поиск по названию"

2. Актуален (is_active)
   - тип: select
   - значения:
     - Да
     - Нет

---

### Actions:

- CreateButton
  - текст: "Создать"
  - переход:
    /dictionaries/colors/new

- ResetFilters
  - сброс всех фильтров

---

## 5.3 Table

### Колонки:

1. Название (name)
2. Актуален (is_active)
3. Дата создания (created_at)

---

### Сортировка

Через COMPONENT: Sorting

Доступна по:

- name
- is_active
- created_at

---

### Поведение

- строка кликабельна

Переход:

/dictionaries/colors/{id}

---

## 5.4 Pagination

- COMPONENT: Pagination
- limit: 20 / 50 / 100
- default: 20

---

# 6. Поведение

---

## 6.1 Загрузка

- GET /colors
- учитывает query state

---

## 6.2 Query параметры

- name
- is_active
- page
- limit
- sort

---

## 6.3 Пустое состояние

- "Данные отсутствуют"

---

## 6.4 Ошибка

- сообщение + retry

---
