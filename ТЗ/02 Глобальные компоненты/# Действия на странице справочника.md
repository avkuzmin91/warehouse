# COMPONENT: Collection Actions

---

# 1. Цель

Реализовать блок действий для страниц списков сущностей.

---

# 2. Контекст использования

Используется на страницах:

- справочник товаров
- справочник клиентов
- любые list-view страницы

---

# 3. Назначение

Содержит действия, относящиеся ко всему списку:

- создание нового элемента
- (расширяемо: экспорт, bulk actions)

---

# 4. Размещение

- над таблицей
- справа
- внутри Filters Panel, правее иконки для сброса фильтров

---

# 5. Состав

## 5.1 Primary Action

- Иконка "+"
- ведёт на страницу создания сущности

---

## 5.2 Secondary Actions

- "Сбросить фильтры"
- очищает Query State (filters)

---

# 6. Поведение

- не содержит бизнес-логики
- не выполняет API запросы
- работает через Query State
- hover-эффект

---

# 7. Ограничения

- не привязан к конкретной сущности
- не содержит таблицу или фильтры
